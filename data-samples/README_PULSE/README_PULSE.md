# PULSE Codex — Operating Rules (Single Source of Truth)

## What this is
This Codex is the official reference for the PULSE project.
If any chat message or code suggestion conflicts with this Codex, the Codex wins.

## Non-negotiables
1) PULSE is population-level early warning (signals), NOT diagnosis and NOT individual clinical decision support.
2) Privacy-by-design: no identifiable patient data in outputs.
3) One global language toggle only (Arabic/English) that changes the whole UI consistently.
4) Explainability: statistical baselines must remain visible (EWMA/CUSUM/Farrington are not hidden).

## How to start any new chat (copy/paste)
اعتمد على PULSE Codex كمرجع حاكم (single source of truth).
لا تغيّر أي قرار معماري/نطاق/لغة إلا بتعديل صريح داخل الكوديكس مع توثيق السبب.

## Change control (Decision Records)
Any meaningful change must be written at the bottom of the relevant Codex file as:
- YYYY-MM-DD — Decision — Reason — Impact
